# Desafio de Controle de Fluxo - DIO

Projeto desenvolvido para o desafio de controle de fluxo da trilha Java Básico da DIO.

## Objetivo
O sistema recebe dois números inteiros via terminal e realiza a impressão no console dos números incrementados, desde que o segundo parâmetro seja maior que o primeiro. Caso contrário, lança uma exceção customizada.

## Como executar
1. Clone o repositório.
2. Navegue até a pasta `src`.
3. Compile com `javac *.java`.
4. Execute com `java Contador`.
